# Otimização dos modelos

- Nesse sprint realizamos uma otimização dos modelos 3D, os tornando mais leves. Dessa forma foi possível construir uma tabela mostrando o antes/depois desse processo.
  
## Metodologia
Utilizando o Blender foi possível realizar:
- O redimensionamento do tamanho dos modelos
- Utilizamos o decimate para reduzir o número de poligonos
- Usamos o Draco para comprimir os modelos

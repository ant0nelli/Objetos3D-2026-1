# Algoritmo Draco

## O que é?

- Draco é uma biblioteca responsável pela compactação e descompactação de malhas geométricas 3D.

## Para que serve?

- Seu objetivo é melhorar o armazenamento e transmissão dos gráficos 3D.
- Por isso ele foi projetado visando a efciência da velocidade de compressão.
